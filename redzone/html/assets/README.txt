=====================================================
REDZONE LEAGUE - Dossier Assets
=====================================================

LOGO:
-----
Placez votre fichier logo.png dans ce dossier.
Le logo sera affiché en haut à gauche de l'interface.

Recommandations:
- Format: PNG avec transparence
- Taille recommandée: 100x100 pixels ou plus
- Fond transparent pour une meilleure intégration

Si le fichier logo.png n'est pas présent, l'espace sera
automatiquement masqué.
